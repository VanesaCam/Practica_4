Modulo 9 — Proyecto Final

Aplicacion Practica en C++

Crea una aplicacion real usando todo lo que aprendiste. Sigue los pasos y entrega tu evidencia para obtener tu certificado.

1
Elige tu aplicacion
Selecciona el tipo de aplicacion que vas a crear. Todas usan clases, objetos y los conceptos de C++ que aprendiste.
2
Instala las herramientas
Necesitas:
a) Compilador g++ — Descarga MinGW-w64 para Windows. Durante la instalacion selecciona la arquitectura x86_64.

b) Editor de codigo — Descarga Visual Studio Code e instala la extension C/C++ de Microsoft.

Verificar instalacion (abrir CMD y escribir): 
g++ --version
# Debe mostrar la version instalada
3
Estructura base del proyecto
Crea un archivo llamado mi_proyecto.cpp con esta estructura:
#include <iostream>
using namespace std;

class MiApp {
public:
    void ejecutar() {
        cout << "Mi aplicacion C++ 2026" << endl;
        // Tu logica aqui
    }
};

int main() {
    MiApp app;
    app.ejecutar();
    return 0;
}
4
Compilar y ejecutar
Abre una terminal (CMD) en la carpeta de tu archivo y ejecuta:
# Compilar
g++ -o mi_proyecto mi_proyecto.cpp
# Ejecutar (Windows)
mi_proyecto.exe
5
Ejemplo: Calculadora con clases
class Calculadora {
private:
    double resultado = 0;
public:
    void sumar(double a, double b) {
        resultado = a + b;
        cout << a << " + " << b << " = " << resultado << endl;
    }
    void restar(double a, double b) {
        resultado = a - b;
        cout << a << " - " << b << " = " << resultado << endl;
    }
    double getResultado() { return resultado; }
};
