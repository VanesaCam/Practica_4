Modulo 3 · Leccion 5
Funciones Recursivas en C++
Una funcion recursiva se llama a si misma. Toda funcion recursiva necesita un caso base que detenga la recursion. Sin el, ocurre un stack overflow.
int factorial(int n) {
  // Caso base
  if (n == 0) return 1;
  // Llamada recursiva
  return n * factorial(n - 1);
}
Importante: Sin caso base → stack overflow.
